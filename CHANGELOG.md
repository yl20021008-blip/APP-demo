# Changelog

## v1.3.4 Font + Audio

本版本优化两个点：

### 1. 页面字体与视觉协调

- 统一全局字体栈；
- 优化标题字重、行距、字距；
- 优化侧边栏、按钮、输入框、指标卡片；
- 优化今日学习单词卡片；
- 表格和状态卡片视觉更统一。

### 2. 单词读音恢复与增强

之前“没有读音”的主要原因是：
- 导入词库只有 `phonetic` 音标，没有真实音频 URL；
- 部分词汇补全逻辑看到已有音标后，不一定继续补音频；
- 今日学习页只在存在 `uk_audio_url/us_audio_url` 时显示播放器。

本版本修复为：
- 今日学习页优先显示词典真实音频；
- 如果没有词典音频，自动显示浏览器 TTS 朗读按钮；
- 词汇补全中心会把缺少音频也视为待补全项目；
- 词汇补全中心新增“缺少音频”统计。

### 部署入口

Streamlit Cloud 的 Main file path 仍然是：

```text
main_app/app.py
```
