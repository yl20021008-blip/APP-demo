# Changelog

## v1.5.0 Morandi Bottom Navigation

本版本优化页面结构和整体色调。

### 1. 常用功能移到底部导航

底部固定导航包含：

```text
首页
今日学习
复习计划
故事记忆
学习统计
```

这些是学习者最常用的操作，进入页面后可以在底部直接切换。

### 2. 不常用功能收纳到左侧栏

左侧栏折叠区包含：

```text
我的词库
上传词库
批量导入词库
词汇补全中心
公共词库管理
```

这些属于词库维护和管理类功能，不再占据主导航。

### 3. 莫兰迪 / 科研风低饱和色系

整体色调改为：

- 米灰背景；
- 鼠尾草绿主色；
- 低饱和棕灰强调色；
- 低对比边框；
- 更柔和的卡片和按钮。

### 部署入口

Streamlit Cloud 的 Main file path 仍然是：

```text
main_app/app.py
```
