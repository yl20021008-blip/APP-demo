# Changelog

## v1.3.0 Story + Example Enhancement

本版本在 v1.2.2 云端保存 + 用户区分版基础上，重点优化两个功能：

### 1. 故事记忆优化

- 故事从原来的固定模板升级为“场景路线式故事”；
- 英文故事中自动加粗目标词；
- 中文故事中嵌入词义线索，帮助建立“词—义—场景”关联；
- 每6个词形成一个场景段落，减少30词故事的机械感；
- 故事结尾自动生成复习小测；
- 新增 provider 标记：`local_narrative_v1_3`；
- 每个学习者仍然独立保存自己的故事。

### 2. 例句自动补充优化

- 优先从词典 API 获取真实英文例句；
- 如果词典没有例句，则自动生成雅思学习例句；
- 支持雅思学术风格 / 日常理解风格；
- 自动生成例句后仍可继续翻译；
- 词汇补全中心新增“词典例句 / 生成例句”统计；
- 生成例句会标记来源：`local_generated_ielts_example`。

### 部署提醒

Streamlit Cloud 的 Main file path 仍然填写：

```text
main_app/app.py
```

不要填写旧的：

```text
app.py
```
