# Changelog

## v1.3.3 UI Status

本版本在 v1.3.2 热修复基础上优化页面状态与操作体验。

### 新增 / 优化

1. **统一状态栏**
   - 首页和各页面顶部显示数据库状态、当前学习者、版本号；
   - 更容易判断是否已连接云端数据库、是否已登录。

2. **侧边栏状态优化**
   - 侧边栏显示当前学习者；
   - 显示数据库模式；
   - 增加“刷新状态”按钮；
   - 保留“退出登录”按钮。

3. **首页操作引导**
   - 登录前提示创建学习者；
   - 登录后根据词库、待学词、到期复习自动显示下一步建议。

4. **页面运行错误提示**
   - 页面内部出错时显示更友好的提示；
   - 保留具体异常信息，方便后续排查。

5. **关键页面状态提示**
   - 上传词库、批量导入、今日学习、词汇补全、故事记忆都增加状态说明；
   - 减少用户不知道当前操作影响范围的问题。

### 部署入口

Streamlit Cloud 的 Main file path 仍然是：

```text
main_app/app.py
```
