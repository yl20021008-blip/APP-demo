v1.3.1 性能优化版部署前必读

1. Main file path 仍然是：
   main_app/app.py

2. 如果旧仓库里还有 pages 文件夹，继续删除。

3. 上传 GitHub 后：
   Commit to main
   Push origin

4. Streamlit Cloud 里：
   Manage app → Reboot app

5. Secrets 保持：
   APP_MODE = "cloud"
   DATABASE_URL = "postgresql+psycopg2://...?...sslmode=require"

6. 补全中心建议：
   每批 10 个
   先关闭自动翻译
