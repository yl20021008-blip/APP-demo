v1.4 公共词库管理版部署前必读

1. 复制本文件夹全部内容到 GitHub 本地仓库根目录。
2. 如果旧仓库里还有 pages 文件夹，删除。
3. GitHub Desktop:
   Commit to main
   Push origin

4. Streamlit Cloud:
   Main file path = main_app/app.py
   Manage app → Reboot app

5. Streamlit Secrets 必须新增：
   ADMIN_PIN = "你自己的管理员PIN"
   PUBLIC_WORDLIST_LOCKED = "true"

6. 使用流程：
   - 管理员登录普通学习者账号；
   - 进入“公共词库管理”；
   - 输入 ADMIN_PIN；
   - 一键重置为预设词库；
   - 普通用户只能学习，不能编辑公共词库。
